# Mapeando Domínio

## Entidades de domínio

- Produto
- Vendas
- Compras
- Fornecedores


## Casos de uso

- Rastrear encomendas
- Visualizar compras
- Visualizar histórico de vendas
- Limite mínimo de produto no estoque
- Alerta de estoque por e-mail e notificação no sistema
- Compra automática para reposição de estoque baseada no fluxo de vendas
- Integração com fornecedores
- Métrica de vendas
